爱记账App
=====
主要功能包括：

(1) 注册登录：用户输入账号密码等信息登录系统，进行相关操作。

(2) 类别管理：对收入和支出类别能够进行添加、修改和删除操作。

(3) 收支管理：记录、修改和删除收入支出的各项信息。

(4) 查询统计：按照一定的周期对收入和支出的信息进行统计。

(5) 个人中心：设置个人信息，
